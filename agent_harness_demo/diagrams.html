① AgentRunner 主闭环

用户输入 → 模型 → 工具 → observation 回喂 → 最终输出，全程记 trace。

flowchart TD A["CLI 输入
命令行接收用户输入"] --> B["load_settings
读取 .env 与环境变量"] B --> C{"USE_REAL_MODEL?
是否启用真实模型"} C -->|false| D["MockModelClient
模拟客户端，无需模型"] C -->|true| E["OpenAICompatibleClient
真实 OpenAI 兼容客户端"] D --> F["AgentRunner.run
启动 Agent 主循环"] E --> F F --> G["start_span(RUN)
开始记录根 trace span"] G --> H["messages = system + user
初始化对话历史"] H --> I{"step ≤ max_steps?
是否未超最大循环次数"} I -->|否| J["end_span error
MaxStepsExceeded 超限结束"] J --> K["返回 AgentResult
带 error 的结果"] I -->|是| L["ModelRequest
组装请求（含工具列表）"] L --> M["model.invoke
调用模型客户端"] M --> N{"调用成功?
是否抛 ModelError"} N -->|ModelError| O["end_span error
记录模型错误并结束"] O --> K N -->|成功| P["record_model_call
记录模型调用 span"] P --> Q{"finish_reason?
模型返回类型"} Q -->|FINAL_ANSWER| R["record_final
记录最终输出 span"] R --> S["end_span ok
正常结束根 span"] S --> T["返回 final_answer
返回最终答案"] Q -->|TOOL_CALL| U["追加 assistant 消息
工具调用决策入历史"] U --> V["遍历 tool_call
逐个处理工具调用"] V --> W["ToolRuntime.execute
执行工具"] W --> X["record_tool_call
记录工具调用 span"] X --> Y["observation 回喂
工具结果作为新消息喂回模型"] Y --> I Q -->|未知| Z["UnknownFinishReason
未知返回类型，报错结束"] Z --> K

② ToolRuntime 执行收口

工具不存在 / 参数错误 / 执行异常都收口成结构化 ToolResult，不抛异常给上层。

flowchart TD A["ToolRuntime.execute
开始执行工具"] --> B["registry.get
从注册表查找工具"] B --> C{"工具存在?
是否已注册"} C -->|否| D["ToolResult ok=False
NOT_FOUND 工具不存在"] C -->|是| E["_validate
按 input_schema 校验参数"] E --> F{"校验通过?
参数是否合法"} F -->|否| G["ToolResult ok=False
VALIDATION_ERROR 参数错误"] F -->|是| H["handler(**validated)
执行工具处理函数"] H --> I{"执行异常?
是否抛错"} I -->|是| J["ToolResult ok=False
EXECUTION_ERROR 执行异常"] I -->|否| K["ToolResult ok=True
成功返回结果"]

③ trace span 树（一次真实任务）

取自运行时的 traces.jsonl，parent_span_id 决定层级。
